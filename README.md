# Willem's dotfiles

### Installation

- Install your SSH keys
- Install Xcode next

```bash
xcode-select --install
```

Clone this repo and install all the things.

```bash
cd ~
git clone https://github.com/willemvb/dotfiles.git .dotfiles
cd .dotfiles
source ./install.sh
```

### Setup maOS defaults

```bash
source ./macos.sh
```

Additionaly, instal the Ayu Light theme in VS Code.
