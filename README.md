# Willem's dotfiles

### Installation
Install Xcode first so we can work with git like normal people

```bash
xcode-select --install
```

Clone this repo and install all the things

```bash
cd ~
git clone git@github.com:willemvb/dotfiles.git .dotfiles
cd .dotfiles
source ./install.sh
```

### Setup maOS defaults

```bash
source ./macos.sh
```
